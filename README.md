# Plugin de calcul des valeurs par défaut dans les formulaires de Géofluence

Le plugin permet de sélectionner une couche de formulaire dans le projet actif et de lancer le calcul des valeurs par défaut.

Seuls les champs pour lesquels une valeur par défaut est défini et pour lesquels la configuraiton "Appliquer la valeur par défaut sur la mise à jour" est activée. 

Une fois exécuter, une fenêtre liste les champs qui ont subis un recalcul. 

Les modifications ne sont pas enregistrés automatiquement. Il appartient à l'utilisateur d'aller valider le recalcul avant d'enregistrer les modifications.
